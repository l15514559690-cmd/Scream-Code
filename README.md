# 尖叫 Code（Scream Code）

终端里的中文 AI 编程助手。

## 下载

用 Git 克隆本仓库，或下载源码 ZIP 后解压到本地文件夹。

## 安装

在**项目根目录**执行：

```bash
bash install.sh
```

## 配置

复制环境变量模板并填写 API 等信息：

```bash
cp .env.example .env
```

用文本编辑器打开 `.env`，按说明填入密钥与模型地址。

## 启动

安装脚本执行完后，按终端提示执行 `source ~/.zshrc`（或脚本给出的配置文件路径）。然后在终端输入：

```bash
scream
```

## 常用斜杠指令（对话里输入）

| 指令 | 说明 |
|------|------|
| `/help` | 查看全部命令 |
| `/memo` | 长效记忆 |
| `/team` | 多代理（群狼）模式 |
| `/new` | 新会话 |
| `/stop` | 中断生成 |
| `/exit` | 退出 |

更完整的列表以界面里 **`/help`** 为准。配置模型也可用 **`scream-config`**。
